# CLI

The extension ships a docs view for the Sculptor CLI.

Useful commands include:

- `sc new`
- `sc dev`
- `sc doctor`
- `sc sync`
- `sc agents`
- `sc agents refresh`
- `sc pkg`
- `sc package`
- `sc reg`
- `sc ureg`
- `sc rm`

The docs are loaded locally from this extension bundle, so they remain available offline.
