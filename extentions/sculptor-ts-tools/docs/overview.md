# SculptorTS Tools

SculptorTS Tools is a VS Code companion for SculptorTS projects.

It helps you:

- detect Sculptor workspaces
- read and navigate the framework documentation
- validate `sculptor.json` and `props.json`
- explore package-aware projects
- jump into command palette actions for common Sculptor flows

## Quick Start

1. Open a workspace that contains `sculptor.json`.
2. The extension activates automatically.
3. Use the `⚒ Sculptor` activity bar entry or the command palette.
