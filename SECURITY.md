# Security Policy

## Supported Versions

SculptorTS is actively maintained. Security fixes are expected on the current release line documented in `readme.md` and `CHANGELOG.md`.

All versions before `v1.0.0` are deprecated and unsupported.
Versions before `v1.0.0` are no longer actively maintained and will not receive future updates.

Current stable support line:

- `@sculptor/core` `v1.0.2`
- `@sculptor/router` `v1.0.2`
- `@sculptor/config` `v1.0.2`
- `@sculptor/paws` `v1.0.2`
- `@sculptor/cli` `v1.0.2`
- `@sculptor/template-registry` `v1.0.2`
- `@sculptor/di` `v1.0.2`

## Reporting a Vulnerability

Do not report security issues in public issues or pull requests.

Report vulnerabilities privately through GitHub Security Advisories or by email to `dev@imprakhartripathi.in` and `imprakhartripathiofficial@gmail.com`.

If maintainers change the reporting channel later, update this file before publishing it.

Include:

- affected package or app area
- version or commit
- impact summary
- reproduction steps
- proof of concept when available

## Response Expectations

After triage, the maintainer team should:

- acknowledge receipt
- confirm whether the report is in scope
- keep the reporter updated while the issue is validated and fixed
- coordinate a release or mitigation before public disclosure

Timeline depends on severity and exploitability, but reports will be handled as soon as practical.

## Responsible Disclosure

- Give maintainers time to validate and patch the issue before public disclosure.
- Avoid sharing exploit details publicly until a fix or mitigation is available.
- If a coordinated disclosure date is needed, agree on it with the maintainers first.

## Security Scope

This policy covers:

- the SculptorTS monorepo
- published packages under `@sculptor/*`
- the CLI and scaffolded output generated by the CLI
- framework runtime behavior, router assembly, config loading, logging, and package registry metadata

This policy does not cover:

- third-party dependencies outside this repository
- custom application code built on top of SculptorTS
- plugins or extensions not maintained in this repository
