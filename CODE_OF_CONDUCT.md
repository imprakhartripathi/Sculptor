# Code of Conduct

## Our Standard

SculptorTS is a public project. We expect contributors, maintainers, and users to treat each other with respect.

We want the project to stay useful, direct, and easy to work in. That means:

- be constructive
- be respectful in disagreement
- stay focused on the code, docs, and issue at hand
- give credit where it is due
- avoid harassment, discrimination, and personal attacks

## Unacceptable Behavior

The following are not acceptable in any project space:

- abusive, insulting, or degrading language
- harassment or repeated unwanted contact
- discriminatory remarks or behavior
- trolling, threats, or intimidation
- sharing private information without permission
- disruptive behavior that makes collaboration harder for others

## Project Spaces

This code of conduct applies to:

- GitHub issues and pull requests
- discussions and reviews
- release coordination
- any other official SculptorTS community space

## Maintainer Responsibility

Maintainers should help keep conversations productive and safe.

When someone crosses the line, maintainers may:

- remind them of expected behavior
- remove or edit harmful content
- pause or close a thread
- restrict participation in project spaces when necessary

## Reporting

If you experience or witness conduct that violates this code, report it through the repository's private security or maintainer contact path.

If a dedicated conduct contact is added later, maintainers should list it here before publishing.

## Scope

This code of conduct covers participation in the SculptorTS project and its official community spaces. It does not replace local laws or platform rules.
